## Projeto LOGIN

Projeto que simula o antigo Orkut, desenvolvido como teste na residência de software da Cubos Academy.

Front-end utiliza React.js e SASS.
Back-end utiliza Node.js e [sdkgen](https://sdkgen.github.io/).
